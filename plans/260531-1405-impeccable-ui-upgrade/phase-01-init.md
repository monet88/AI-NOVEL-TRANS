# Phase 1: Init

## Overview
Khởi tạo ngữ cảnh thiết kế Impeccable cho dự án bằng cách tạo PRODUCT.md và DESIGN.md để AI nắm rõ định hướng giao diện của ứng dụng AI Novel Weaver.

## Implementation Steps
1. Chạy lệnh khởi tạo `/impeccable init` (hoặc chạy qua CLI nếu có) để AI Agent tự động thu thập định hướng thiết kế.
2. Tạo hoặc bổ sung thông tin cho file `PRODUCT.md` ở thư mục gốc để mô tả đúng các tính năng dịch thuật truyện và tập đối tượng người dùng.
3. Tạo hoặc bổ sung file `DESIGN.md` để định vị bảng màu (sắc độ tối), hệ font chữ (font-pair, tỷ lệ chữ dễ đọc truyện), và các nguyên tắc thiết kế chuyển động (motion principles).

## Success Criteria
- [ ] File `PRODUCT.md` mô tả đúng nghiệp vụ dịch thuật tiểu thuyết dài tập của ứng dụng.
- [ ] File `DESIGN.md` được tạo chứa các quy chuẩn Typography, bảng màu Tinted Dark, và Spacing.
